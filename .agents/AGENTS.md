# UI and Data Consistency
When provided with a UI mockup/screenshot, do NOT blindly copy static data or mock values (e.g., hardcoded prices, tax percentages, or English labels if the app is in Indonesian). ALWAYS cross-reference with related pages, database schemas, and actual data flows to ensure mathematical and logical consistency across the application.
